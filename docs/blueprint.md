# **App Name**: DigiCar

## Core Features:

- Car Catalog: Browse a comprehensive catalog of available cars.
- Virtual Assistant: Chat with an AI assistant to get personalized car recommendations, integrating available data about the different models as a tool.
- Car Comparison: Compare different car models side-by-side.
- Car Simulator: Use a car simulator to get personalized recommendations for different scenarios.
- Lead Capture: Capture user information for sales and follow-up.

## Style Guidelines:

- Primary color: Warm gold (#D4A24E) to convey trust, history and prestige.
- Background color: Light beige (#F5F5DC), of the same hue family as the primary, and significantly desaturated to serve as a neutral background that allows other colors to stand out.
- Accent color: Soft brown (#A0785A) to complement the gold.
- Body and headline font: 'PT Sans', a modern sans-serif.
- Sleek and modern icons that are brand-appropriate.
- Clean and spacious layout for ease of navigation.
- Subtle transitions and animations for a premium feel.
